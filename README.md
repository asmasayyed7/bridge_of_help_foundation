# Bridge Of Help Foundation Website

A responsive website for Bridge Of Help Foundation, an NGO dedicated to supporting underprivileged children and families through food programs, volunteer activities, and community initiatives.

## Features

- **Responsive Design**: Mobile-first approach with breakpoints for tablets and mobile devices
- **Modern UI**: Clean and professional interface with custom styling
- **Sections**:
  - Hero section with statistics
  - About Us section
  - Activities/Projects showcase
  - Volunteer opportunities
  - Footer with contact information

## Getting Started

1. Clone this repository:
   ```bash
   git clone <your-repository-url>
   ```

2. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

3. Open `index.html` in your web browser to view the website.
